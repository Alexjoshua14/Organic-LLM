# Noesis

Noesis is a chat experience for exploring a topic — starter prompts, sparks along the thread, and suggested replies drawn from how you have talked before.

## What it is for

- Surfacing what you might want to discuss next on a theme
- Topic threads with tailored openers instead of a blank composer
- **Suggest my reply** — draft a reply from context, then edit before sending
- Steer notes — lightweight nudges that shape where the thread goes

## How it differs from main Chat

Noesis is built around exploration, not general Q&A. The UI emphasizes prompts and reply assistance; scroll position is restored when you return to a thread mid-conversation.

## When to use Noesis

Open Noesis when you have a topic in mind but want help getting started, or when you want suggested replies without leaving the thread. Use main Chat for open-ended work that is not topic-scoped.

# Remy

Remy is a dedicated assistant surface — threads with the Remy persona, separate from main Chat.

## What it is for

- Conversations styled for Remy’s voice and behavior
- A focused lane when you want that assistant specifically
- Same encryption and workspace context as other surfaces

## How it differs from Chat

Remy is a distinct assistant entry point, not a mode toggle inside main Chat. Switch from the sidebar rail when you want Remy rather than the default assistant.

## When to use Remy

Open Remy when the task fits that assistant’s role or tone. Use main Chat (or Arcadia / Noesis) for general workspace work.
